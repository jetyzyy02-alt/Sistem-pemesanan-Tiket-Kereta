# Sistem-pemesanan-Tiket-Kereta
Sistem Pemesanan Tiket Kereta Api berbasis Python dengan database Flat File (CSV). Aplikasi ini menerapkan operasi CRUD (Create, Read, Update, Delete), Linear Search, Bubble Sort, serta struktur data Linked List dan Queue. Data pemesanan disimpan dalam file CSV dan dikelola melalui antarmuka Command Line Interface (CLI).

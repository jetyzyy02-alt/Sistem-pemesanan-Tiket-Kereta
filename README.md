# Sistem-pemesanan-Tiket-Kereta
Muhamad Sahrul_1127

Sistem Pemesanan Tiket Kereta Api adalah aplikasi berbasis Python yang digunakan untuk mengelola data pemesanan tiket kereta api. Aplikasi ini menerapkan konsep struktur data dan algoritma untuk melakukan pengelolaan data secara efisien, dengan penyimpanan data menggunakan file CSV sebagai database flat file. Pengguna dapat melakkukan pemesanan, mencari, melihat, mengubah, menghapus, dan mengurutkan data pemesanan melalui antarmuka command line Interface (CLI)

Tujuan Proyek
1. Menerapkan konsep data dalam pengembangan aplikasi
2. Mengimplementasikan operasi CRUD (Create, Read, Update, Delete)
3. Mengelola data pemesanan tiket menggunakan file CSV sebagai database
4. Mengimplementasikan algoritma pencarian (Searching) dan pengurutan (Sorting)
5. Mensimulasikan sistem antrean pemesanan tiket menggunakan Queue
6. Membuat aplikasi pemesanan tiket kereta api berbasis CLI yang sederhana dan mudah digunakan

Fitur Utama
1. Pemesanan tiket kereta api
2. Menampilkan seluruh data pemesanan
3. Mengubah data pemesanan tiket
4. Membatalkan atau menghapus pemesanan tiket
5. Mencari data tiket berdasarkan ID Tiket
6. Mengurutkan data penumpang berdasarkan nama
7. Mengelola antrean pemesan tiket
8. Menyimpan dan membaca data dari file CSV

Struktur Data Yang Digunakan
1. Linked List
    Digunakan untuk menyimpan dan mengelola data pemesanan tiket selama program berjalan.
Fungsi:
    1. Menambah data tiket.
    2. Menampilkan data tiket.
    3. Mencari data tiket.
    4. Menghapus data tiket.
2. Queue
   Digunakan untuk mensimulasikan antrean loket pemesanan tiket
   fungsi:
     1. Menambahkan penumpang ke antrean (Enqueue)
     2. Melayani penumpang dalam antrean (Dequeue)

Algoritma Yang Digunakan
1. Linear Search
   Digunakan untuk mencari data berdasarkan ID tiket
   Cara Kerja:
   1). Menelusuri data satu persatu hingga data ditemukan
   
3. Bubble Sort
   Digunakan untuk mengurutkan data penumpang berdasarkan nama
   Cara Kerja:
   1). Membandingkan dua data yang berdekatan
   2). Menukar posisi jika urutannya tidak sesuai
   3). Diulang hingga seluruh data terurut

Menu Program
1. Pesan Tiket
2. Lihat Data Pemesanan
3. Update Tiket
4. Batalkan Tiket
5. Cari Tiket
6. Sorting Nama Penumpang
7. Tambah Antrean
8. Layani Antrean
9. Keluar
